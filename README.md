# 🚴 Finnish for Cyclists

A beautiful, interactive flashcard app for learning essential Finnish vocabulary for bicycle touring and camping in Finland.

## Features

- **130 Finnish words and phrases** organized into 11 categories
- **Interactive flashcards** with flip animation
- **Category filtering** - Focus on specific topics
- **Progress tracking** with visual progress bar
- **Keyboard navigation** - Arrow keys, spacebar, and number keys
- **Responsive design** - Works on desktop and mobile
- **No dependencies** - Pure HTML, CSS, and JavaScript

## Categories

1. **Greetings** - Hello, goodbye, welcome
2. **Basics** - Yes, no, please, thank you
3. **Directions** - Left, right, straight ahead
4. **Cycling** - Bike parts, repairs, routes
5. **Camping** - Tent, fire, camping sites
6. **Food & Drink** - Restaurant, shopping, meals
7. **Accommodation** - Hotels, camping, booking
8. **Shopping** - Prices, payment, stores
9. **Emergency** - Help, police, hospital
10. **Social** - Conversations, introductions
11. **Nature** - Forest, lake, weather

## How to Use

### Local Development
Simply open `index.html` in your browser. No server required!

### Keyboard Shortcuts
- **← →** - Navigate cards
- **Space** - Flip card
- **1-9, 0** - Jump to categories

## Deploy to GitHub Pages

This app is ready to deploy to GitHub Pages as a static website:

1. Create a new repository on GitHub
2. Add the remote and push:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git branch -M main
   git push -u origin main
   ```
3. Go to **Settings → Pages** in your repository
4. Select **main** branch as the source
5. Your app will be live at `https://YOUR_USERNAME.github.io/YOUR_REPO/`
